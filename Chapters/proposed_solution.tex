\section{Local Planner Waypoints}

The result of the global search is a sequence of intersections the robot should follow. To get the path polyline (a sequence of waypoints $W=\{w_1, w_2, \dots, w_n\}$ connected by straight segments), each edge is expanded using OSM data.
